The official Ruby library for interacting with the "RESTful API":http://mite.yo.lk/api of "mite":http://mite.yo.lk/, a sleek time tracking webapp.

h3. Install
   
   sudo gem install mite-rb

mite-rb requires activeresource and activesupport gems to be installed.

h3. Documentation

Take a look at lib/mite-rb.rb for examples and documentation.

h3. Use the console to interact with the mite.api

   irb -r lib/mite-rb/console
